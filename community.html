공동체 모델

퇴직 후 하강이 아닌,
상승하는 삶의 구조

스스로 노후 복지를 적립하는 자립형 순환 모델. 지자체 예산을 최소화하는 새로운 대안입니다.

왜 달라야 하는가

기존 모델과의 근본적인 차이

구분 기존 모델 (퇴직 후 하강 경로) 자연돌봄 모델 (상승 경로)
사회적 위치 청년도 노인도 아닌 정책 사각지대. 지원도 활용도 없는 무역할 공백기 공동체의 핵심 운영 주체. 직원 겸 조합원 — 기여자로 재정의
퇴직 후 경로 자유 → 심심함 → 고립 → 건강 악화 → 무력감 → 요양원 공동체 참여 → 봉사·생산·돌봄 → 존엄한 노년까지 상승
보유 자산 활용 30~40년 경험·기술·인간관계가 방치됨 지역사회 돌봄 인프라로 전환. 청년 멘토·아동 교육·노인 돌봄에 기여
노후 불안 불확실한 노년. 요양원 입소 = 사회적 역할 종료 공동체 안에서 자신의 노후를 직접 설계. 임종까지 익숙한 사람들 곁에서
재정 구조 국민연금·퇴직금 소진. 외부 보조금 의존 출자금·돌봄코인 적립 → 입주비 자동 전환. 스마트팜·태양광 자체 수익
지역사회 기여 귀촌 소비자 역할에 그침 인구소멸 지역 활력 창출. 폐교 재생, 아동 양육, 산림 관리까지 담당
생애주기 연계 모델

3단계 생애주기 프로세스

55세에 참여해서 임종까지 — 하나의 공동체 안에서 모든 생애 단계를 완성합니다.

1단계 · 진입기

핵심세대 — 공동체의 엔진

만 55~64세 · 출퇴근 또는 상주

30~40년 전문 역량을 공동체 자산으로 활용합니다. 출자금(1좌 10만원, 최소 10좌)을 납입하면 향후 입주 보증금으로 자동 전환됩니다. 지자체 신중년 일자리 보조금도 연계됩니다.

출자금 납입 돌봄코인 적립 현금 급여 신중년 보조금
2단계 · 전환기

파트너세대 — 지혜의 돛

만 65~75세+ · 12평형 독립주거동 입주

적립한 출자금·코인으로 입주 보증금을 충당합니다. 월 170만원 ALL-IN으로 식사 3끼·세탁·청소·차량·프로그램·기본의료가 포함됩니다. 아동 양육 지원·유기견 사회화·청년 멘토링 역할을 이어갑니다.

월 170만원 ALL-IN 일 3시간 유급 독립 주거 국민연금 연계
3단계 · 돌봄기

돌봄 수혜자

만 75세 이후 · 케어동 이전

외부 요양원이 아닌, 25~30년 함께한 사람들 곁에서 돌봄을 받습니다. 누적한 돌봄코인으로 요양비를 충당하고, 장기요양보험과 연계하여 사망 시까지 공동체 안에서 생활합니다.

코인 소진 장기요양 연계 케어동 입주 존엄한 임종
입주 유형

3가지 입주 타입

건강 상태와 필요에 따라 선택할 수 있는 3가지 입주 유형을 제공합니다. 공동체 안에서 자연스럽게 유형 간 전환이 이루어집니다.

TYPE A

독립 생활형

월 170만원
보증금 2,000만원 + 월 170만원 ALL-IN
  • 식사 3끼 제공 (유기농 스마트팜 식재료)
  • 세탁·청소·차량 서비스 포함
  • 프로그램·커뮤니티 활동 참여
  • 기본의료 (간호사 상주)
  • 유기견 공동 돌봄 프로그램
  • 스마트팜 체험·마을학교 참여
TYPE B

경증 요양형

월 90만원
장기요양급여(4~5등급) + 본인부담 90만원
  • Type A 서비스 전체 포함
  • 장기요양보험 4~5등급 수급자
  • 일상 생활 지원 서비스
  • 요양보호사 상주
  • 건강 모니터링 강화
  • 공단 급여분 공동체 직수령
TYPE C

중증 케어형

월 60만원
장기요양급여(1~3등급) + 본인부담 60만원
  • 24시간 전문 요양 서비스
  • 장기요양보험 1~3등급 수급자
  • 케어동 전용 병실 입주
  • 전문 간병인 배치
  • 가족 면회·참여 프로그램
  • 임종 케어 프로그램 제공

※ 장기요양보험 등급 인정률 약 82%(공단 통계 기준), v9 재무모델은 보수적 80% 수급률 적용
